# 数据可视化交互平台

一个功能完整的现代化数据可视化交互平台，支持多种图表类型、主题切换、实时数据编辑和响应式布局设计。

## 🌟 主要功能

### 🎨 主题系统
- **明亮主题**：清爽的白色界面，适合日间使用
- **深色主题**：护眼的深色界面，适合夜间使用
- **主题持久化**：自动保存用户主题偏好
- **平滑切换**：主题切换带有优雅的过渡动画

### 📊 图表类型
- **柱状图**：数据标签显示在柱状图上方，便于对比
- **折线图**：优化的标签样式，清晰展示趋势
- **饼图**：显示数值和百分比，直观展示比例
- **环形图**：现代化的环形数据展示
- **雷达图**：多维度数据对比分析

### 🎨 配色方案
- **默认配色**：活力四彩配色方案
- **莫兰迪色系**：优雅低饱和度配色
- **冰川色系**：清冷蓝色系配色
- **日落色系**：温暖橙红色系配色
- **森林色系**：自然绿色系配色
- **薰衣草色系**：浪漫紫色系配色

### 📈 数据管理
- **多数据集**：内置4个不同领域的示例数据集
- **实时编辑**：通过模态框直接编辑数据值
- **即时更新**：数据修改后图表和统计信息同步更新
- **数据导出**：支持图表图片下载功能

### 📋 布局设计
- **垂直布局**：所有板块按垂直方向排列，保持界面整洁
- **响应式设计**：完美适配各种屏幕尺寸
- **卡片式设计**：现代化的圆角卡片布局
- **毛玻璃效果**：背景采用半透明毛玻璃设计

## 📊 数据集内容

### 1. 图书采购分析
展示不同地区在各类图书采购中的对比情况
- **数据维度**：家庭、小说、心理、科技、儿童、教育
- **地区对比**：地区1 vs 地区2
- **分析价值**：了解地区间采购偏好差异

### 2. 汇率走势分析
展示不同时间点的汇率变化趋势
- **时间维度**：月度数据（1日-30日）
- **历史对比**：2017年7月 vs 2019年7月
- **分析价值**：观察汇率波动规律

### 3. 销售数据分析
展示月度销售额的变化趋势
- **时间范围**：1月-6月
- **数据指标**：销售额（万元）
- **分析价值**：了解销售业绩和季节性变化

### 4. 用户分布统计
展示不同地区用户分布情况
- **地理分布**：华北、华东、华南、西南、西北、东北
- **用户数量**：各地区用户统计
- **分析价值**：了解用户地理分布特征

## 🎯 界面布局

### 垂直排列顺序（从上到下）
1. **图表样式控制** - 图表类型、配色方案、动画设置
2. **图表容器** - 主要的数据可视化展示区域
3. **统计信息** - 数据总量、平均值、最大值、最小值
4. **图表说明** - 当前图表的标题和详细描述
5. **数据面板** - 数据表格和编辑功能

### 数据标签特性
- **柱状图**：标签显示在柱状图上方，白色背景
- **折线图**：优化的标签样式，位于数据点附近
- **饼图/环形图**：显示数值和百分比，居中对齐
- **雷达图**：简洁的标签样式，保持界面整洁

## 🔧 技术架构

### 前端技术栈
- **HTML5**：语义化标记，无障碍访问支持
- **CSS3**：现代布局（Grid/Flexbox）、动画、变量
- **JavaScript ES6+**：模块化编程、异步处理
- **Chart.js**：专业的数据可视化库
- **ChartDataLabels**：图表数据标签插件

### 核心功能实现
- **主题系统**：CSS变量 + JavaScript动态切换
- **响应式布局**：CSS媒体查询 + 弹性布局
- **数据管理**：JavaScript对象 + 本地存储
- **图表渲染**：Chart.js API + 自定义配置
- **交互处理**：事件监听 + 状态管理
